# Fantasy Junior A - App Oficial

Aplicación de gestión gamificada para el equipo Junior A del Fundación CB Granada.

## Repositorio Oficial
Este proyecto está alojado en: [Fantasy-Junior-A-Final](https://github.com/lucasmaqalv/Fantasy-Junior-A-Final)

## ⚠️ IMPORTANTE PARA DESPLIEGUE EN NETLIFY

Si has descargado esto como un ZIP:
1. **NO subas el archivo .zip a GitHub.**
2. Descomprime el zip en tu ordenador.
3. Sube **los archivos extraídos** (src, public, index.html, package.json, etc.) a la raíz del repositorio de GitHub.

## Características
- **Gamificación**: Sistema de XP, niveles (Arenas) y moneda virtual.
- **Gestión de Equipo**: Perfiles de jugadores, estadísticas y roles.
- **Pizarra Táctica**: Playbook digital y cuestionarios.
- **Base de Datos**: Conexión a Supabase para persistencia en tiempo real.
- **PWA**: Instalable en móviles como App Nativa.

## Configuración Técnica
- Build Command: `npm run build`
- Publish Directory: `dist`
- Framework: Vite + React
